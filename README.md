### 🚀 Projetos em Destaque

#### 🔬 [Epidemiologia SINAN](https://github.com/GabrielMaia2514/epidemiologia-sinan)
Um motor de ingestão de dados em larga escala para o sistema de saúde pública brasileiro.
* **O desafio:** Processar arquivos massivos (CSV/ZIP) de forma eficiente.
* **A solução:** Implementação de processamento não-bloqueante (`@Async`), streaming de dados para economia de memória e lógica de *Upsert* para garantir integridade.
* **Tech Stack:** Java 17, Spring Boot, OpenCSV, PostgreSQL.

#### 🌳 [JSON Tree Viewer](https://github.com/GabrielMaia2514/json-tree-viewer)
Componente Angular especializado para visualização hierárquica de estruturas de dados.
* **O desafio:** Criar uma visualização recursiva e intuitiva para objetos complexos.
* **Diferenciais:** Suporte a profundidade dinâmica, cópia para clipboard e alta performance em objetos grandes.
* **Tech Stack:** Angular, TypeScript, SCSS.
